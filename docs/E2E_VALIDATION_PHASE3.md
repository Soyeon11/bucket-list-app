# Phase 3 E2E 검증 체크리스트

> 작성일: 2026-05-10  
> 검증 기준: 실제 기기(iOS Expo Go) + FastAPI 서버 로컬 실행 환경  
> 기록 방법: 각 항목 결과에 ✅ (통과) / ❌ (실패) / ⏭️ (스킵) 기입

---

## 사전 준비

- [ ] FastAPI 서버 실행 확인: `http://localhost:8000/health` → `{"status": "ok"}`
- [ ] Expo Metro 번들러 실행 확인: `http://localhost:8081`
- [ ] iOS Expo Go 앱 실행 후 QR 코드 스캔
- [ ] Supabase에 테스트 계정 로그인 상태 확인
- [ ] **active 상태의 버킷리스트 아이템 1개 이상 존재 확인**
- [ ] **Supabase Storage 버킷(`activity-media`) 생성 및 접근 권한 설정 확인**
